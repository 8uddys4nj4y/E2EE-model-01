# E2EE Demo 

Demo showing how end-to-end encryption works – exactly like WhatsApp & Signal.

- 100% offline (no internet needed)
- Real RSA encryption/decryption
- Works in any browser
- Perfect for teaching

## Files
- `index.html` – Main page
- `style.css`  – WhatsApp-style look
- `script.js`  – All the magic (60 lines)

## How to Run
1. Download all 3 files in one folder
2. Double-click `index.html`
3. Open in two tabs side-by-side
4. Type & send messages → watch encryption live!

## Features
- Encrypted gibberish shown in both chats
- Only receiver can read the real message
- Console logs (F12) show encrypted & decrypted text
- Pure JavaScript – no server, no libraries
